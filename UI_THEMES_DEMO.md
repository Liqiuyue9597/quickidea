# QuickIdea UI 风格 Demo

我为你创建了 **4 种不同的 UI 风格**，每种都有完整的配色方案和视觉效果。

## 🎨 四种风格预览

### 风格 A: 极简黑白 (Minimal)

**设计理念**: 极致简约，黑白灰配色，强调内容本身

**配色方案**:
- 背景: 纯白 `#FFFFFF`
- 次要背景: 浅灰 `#F8F8F8`
- 主文字: 黑色 `#000000`
- 强调色: 黑色 `#000000`
- 标签色: 灰度渐变 `#000000` → `#999999`

**视觉特点**:
- 无彩色，纯黑白
- 细线分隔 `#E5E5E5`
- 极浅阴影（几乎看不见）
- 类似: Things, Clear

**适合场景**: 专注内容，减少干扰，专业感

---

### 风格 B: 柔和渐变 (Gradient)

**设计理念**: 温柔舒适，浅色渐变，柔和过渡

**配色方案**:
- 背景: 浅蓝灰 `#F5F7FA`
- 次要背景: 柔和灰蓝 `#E8ECF1`
- 主文字: 深蓝灰 `#2C3E50`
- 强调色: 天蓝色 `#5DADE2`
- 标签色: 紫、粉、黄、紫 `#A29BFE` `#FD79A8` `#FDCB6E` `#6C5CE7`

**视觉特点**:
- 柔和的蓝灰色调
- 渐变感的标签颜色
- 淡淡的蓝色阴影
- 类似: Notion, Craft

**适合场景**: 长时间使用，护眼舒适

---

### 风格 C: 活力多彩 (Vibrant)

**设计理念**: 充满活力，鲜艳明快，年轻化

**配色方案**:
- 背景: 纯白 `#FFFFFF`
- 次要背景: 浅粉 `#FFF5F5`
- 主文字: 深灰 `#1A1A1A`
- 强调色: 珊瑚红 `#FF6B6B`
- 标签色: 红、青、黄、绿 `#FF6B6B` `#4ECDC4` `#FFE66D` `#A8E6CF`

**视觉特点**:
- 鲜艳的彩色标签
- 轻微的粉色背景
- 彩色阴影效果
- 类似: Notion Calendar, Any.do

**适合场景**: 活跃氛围，激发创意

---

### 风格 D: 复古纸质 (Paper)

**设计理念**: 温暖复古，纸张质感，手账风格

**配色方案**:
- 背景: 米黄色 `#F4F1E8`（像旧纸）
- 次要背景: 暖灰 `#EBE7DC`
- 卡片背景: 象牙白 `#FFFEF9`
- 主文字: 深棕 `#3E3B32`
- 强调色: 咖啡色 `#B8956A`
- 标签色: 棕色系 `#B8956A` `#8B7355` `#A67C52` `#6B5D4F`

**视觉特点**:
- 温暖的米黄底色
- 棕色系标签
- 纸张质感
- 类似: Bear, Day One

**适合场景**: 温馨氛围，记录生活

## 📱 如何查看 Demo

### 方法 1: 在项目中添加主题系统（推荐）

我已经创建了文件：
- `Theme.swift` - 完整的主题系统
- `ThemePreviewView.swift` - 可交互的预览页面

需要在 Xcode 中：
1. 右键 QuickIdea 文件夹
2. "Add Files to QuickIdea..."
3. 选择这两个文件
4. 勾选 QuickIdea target

然后运行 App，会有一个"主题"标签页，可以实时切换查看。

### 方法 2: 查看代码示例

所有配色都在 `Theme.swift` 中定义，你可以直接看到每个颜色的 Hex 值。
